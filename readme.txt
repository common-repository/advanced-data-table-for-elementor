The easiest and lightweight elementor Advanced Data Table.

== Description ==
Advanced Data Table for Elementor, the must-have plugin for Elementor users looking to effortlessly create stunning and functional tables on their WordPress websites. Elementor table allowing you to build and customize tables directly within the familiar Elementor interface. With live customization, you can see your changes in real-time, ensuring every detail matches your vision perfectly.


= Requirements =
* PHP 7.0
* Elementor


== Installation ==
1. Install Advanced Data Table plugin
2. Make sure you also have Elementor activated.
